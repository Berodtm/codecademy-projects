Rob presents Snake 2

Inspired by the Nokia 3310 I have styled this game to mimic the iconic game released in Q4 2000.

Controls and Instructions

The snake will go in a streight line until the user turns the snake. The object of the game is to eat the apple by passing through it.

The Game ends when the snake hits the wall or itself.

Use the arrows on your keyboard to move the snake.
